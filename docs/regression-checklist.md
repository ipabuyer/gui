# 最小回归清单

## 自动校验
- 运行 `powershell -ExecutionPolicy Bypass -File .\scripts\smoke-check.ps1 -Configuration Debug`
- 预期结果：
  - MSIX `Name/Publisher/Version/Language` 校验通过
  - `x64` 与 `ARM64` 构建均成功
  - 两个平台产物中都存在 `ipatool.exe`

## 手工回归
1. 登录
- 使用测试账号 `test/test` 登录
- 验证：登录成功、状态提示正常、可返回主页

2. 搜索
- 在标题栏输入应用名并搜索
- 验证：列表有结果，字段显示完整（名称/APPID/开发者/版本/价格）

3. 购买
- 选择免费应用点击“购买”
- 验证：状态更新为“已购买”或“已拥有”，筛选结果同步

4. 下载
- 从主页加入下载队列
- 在下载页执行“开始下载队列”
- 验证：状态流转正确（待下载/下载中/成功/失败/已终止），日志可复制/清空/终止当前下载

5. 设置持久化
- 修改国家代码并保存
- 修改下载目录并保存
- 重启应用后验证设置仍生效
